# Como Contribuir com o GovData-BR

Obrigado pelo interesse em contribuir! Este projeto é open source e toda ajuda é bem-vinda. 🎉

## 🐛 Reportando Bugs

Encontrou algo errado? Abra uma [Issue](../../issues/new) com:
- Descrição clara do problema
- Passos para reproduzir
- O que era esperado vs o que aconteceu
- Prints ou logs se possível

## 💡 Sugerindo Funcionalidades

Tem uma ideia? Abra uma [Issue](../../issues/new) com:
- Descrição da funcionalidade
- Por que seria útil para o projeto
- Exemplos visuais se tiver

## 🔀 Fazendo uma Contribuição

1. Faça um **fork** do repositório
2. Crie uma branch descritiva: `git checkout -b feat/minha-feature`
3. Faça suas alterações
4. Commit seguindo o padrão abaixo
5. Abra um **Pull Request** para a branch `main`

## 📝 Padrão de Commits

Use o padrão [Conventional Commits](https://www.conventionalcommits.org/):
```
feat: adiciona novo indicador econômico
fix: corrige carregamento do mapa
docs: atualiza README
refactor: reorganiza estrutura de pastas
```

## 🤝 Código de Conduta

- Seja respeitoso e inclusivo
- Críticas são bem-vindas, ataques pessoais não
- Dúvidas? Abra uma Issue sem hesitar
- Toda contribuição, grande ou pequena, é valorizada
